# 老干妈工作计时法

> 本项目是一个整合体验 react-ts-antd 的项目 [项目预览链接](https://hentaitang.github.io/Tomato-ts)

## 本项目主要用到的技术栈

- react
- react-router
- react-redux
- ant-design
- typescript
- scss
- axios

## 实现功能

1. 用户登录注册
2. 25 分钟定时闹钟
3. todolist 待办任务列表
4. 数据统计(删改)
